Burger Builder App

#React

#Axios

#React-Router

#Redux

#Firebase

View Project

The Burger Builder app is built using React with Redux, React Router v4, Axios, and Firebase for: Hosting, Authentication, and the Database. Customers must first Authenticate themselves by creating an account, which is stored in the firebase database. Afterwards customers can create multiple orders by customizing their burgers. Once satisfied, customers can place an order which brings up an order form which is stored in the database as well. The authentication links the users orders to their address.

Yelp Camp

#Node.js

#Express

#MongoDB

#HTML

#CSS

#Passport

View Project

Yelp Camp is an a review site built using Node.js as the server side code. Express is used for the routing component and data is stored using MongoDB. Users can upload new campsites with rates/night and anyone can view all uploaded campsite thumbnails on the home page. However, to be able to view the full postings, you need to be authenticated, which was done with the #Passport library. Once authenticated, users can view full postings, post their own campsites, and leave comments, all of which are stored using MongoDB.

SmartBrain API

#React

#Redux

#Node.js

#Express

#PostgresSQL

#HTML5

View App Project View Server Project

Smartbrain Api was built using #React and #Redux. Users must sign up, and then are prompted to upload photos of people so that the api can use facial recognition to show its ability to track faces. The facial recognition is mapped by using the Clarifai api and drawn in using CSS borders. The app stores users and their data via the #PostgresSQL database. The server is coded in #Node.js with #Express for the routing.
